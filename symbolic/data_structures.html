Analysis-related data structures¶

The data structures used for symbolic analysis are meant to serve as black-boxes for the vast majority of users. Nevertheless, they are briefly documented here for the sake of posterity.

DistSeparatorTree¶

type struct LocalSepOrLeaf¶
int parent¶

The index of the local parent separator (\(-1\) if it does not exist).

int offset¶

The first reordered index of this node.

std::vector<int> indices¶

The original indices of this node.

type struct DistSeparator¶
mpi::Comm comm¶

The communicator for this distributed separator.

int offset¶

The first reordered index of this separator.

std::vector<int> indices¶

The original indices of this separator.

type struct DistSeparatorTree¶
std::vector<LocalSepOrLeaf*> localSepsAndLeaves¶

An array of pointers to local separators and leaves.

std::vector<DistSeparator> distSeps¶

An array of distributed separators.

Note

This array does not include the single process separator/leaf.
