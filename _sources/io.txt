Input/output
************

Print
-----

.. cpp:function:: void Print( const DistGraph& graph, std::string msg="DistGraph" )

   Prints the edges in the graph from the root process.

.. cpp:function:: void Print( const DistSparseMatrix<T>& A, std::string msg="DistSparseMatrix" )

   Prints the nonzero triplets in the sparse matrix from the root process.
