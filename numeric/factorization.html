Factorization¶

LDL factorization¶

void LDL(DistSymmInfo& info, DistSymmFrontTree<F>& L, FrontType newFrontType)¶

Performs either an \(LDL^T\) or \(LDL^H\) factorization, depending on whether L is marked as Hermitian. See tests/VectorSolve for an example usage.

Note

This routine does not pivot, so it should be used with caution on matrices which are not Hermitian positive-definite.

Data structures¶

Note

The following data structure is meant to be treated as a black-box by the vast majority of users, but its member variables are documented here for users who are interested.

DistSymmFrontTree¶

This data structure represents a distributed symmetric \(LDL^T\) or \(LDL^H\) factorization.

type struct LocalSymmFront<T>¶
Matrix<T> frontL¶

The left half of a local symmetric frontal matrix. If frontL is \(m \times n\), then the associated node has \(n\) vertices and the lower structure is of size \(m-n\).

mutable Matrix<T> work¶

Used for temporary results during operations with this frontal matrix.

type struct DistSymmFront<T>¶
mutable DistMatrix<T, VC, STAR> front1dL¶

The left half of a frontal matrix in a 1D distribution. If front1dL is \(m \times n\), then the associated node has \(n\) vertices and the lower structure is of size \(m-n\).

mutable DistMatrix<T, VC, STAR> work1d¶

Used for temporary results which are in a 1D distribution.

mutable DistMatrix<T> front2dL¶

The left half of a frontal matrix in a 2D distribution.

mutable DistMatrix<T> work2d¶

Used for temporary results which are in a 2D distribution.

DistMatrix<T, VC, STAR> diag¶

Used for storing the diagonal of the frontal matrix.

type enum FrontType¶

Can be set to either

  • SYMM_1D: Symmetric/Hermitian fronts distributed in a 1D manner
  • SYMM_2D: Symmetric/Hermitian fronts distributed in a 2D manner
  • LDL_1D: LDL factorization distributed in a 1D manner
  • LDL_2D: LDL factorization distributed in a 2D manner
  • LDL_SELINV_1D: LDL factorization with inverted diagonal blocks distributed in a 1D manner
  • LDL_SELINV_2D: LDL factorization with inverted diagonal blocks distributed in a 2D manner
type struct DistSymmFrontTree<T>¶
bool isHermitian¶

If true, the matrix is assumed to be Hermitian; otherwise, it is treated as symmetric.

FrontType frontType¶

Specifies the form of the frontal matrices.

std::vector<LocalSymmFront<T>> localFronts¶

The vector of local frontal matrices.

std::vector<DistSymmFront<T>> distFronts¶

The vector of distributed frontal matrices.

DistSymmFrontTree(Orientation orientation, const DistSparseMatrix<T>& A, const DistMap& map, const DistSeparatorTree& sepTree, const DistSymmInfo& info)¶

A constructor which converts a distributed sparse matrix into a symmetric frontal tree which is ready for factorization (e.g., with LDL() or BlockLDL()).

type struct DistSymmFrontTree<F>¶
