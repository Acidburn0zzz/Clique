The MultiVector class¶

The MultiVector<T> class is the standard interface in Clique for setting up multiple right-hand sides for sequential solves: it is a simplified version of DistMultiVector<T>.

type class MultiVector<T>¶

MultiVector()¶

Constructs a single vector of length zero.

MultiVector(int height, int width)¶

Constructs width vectors, each of the specified height.
