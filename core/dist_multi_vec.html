The DistMultiVec class¶

The DistMultiVec<T> class is the standard interface in Clique for setting up several right-hand sides for solves: it uses the same simple one-dimensional distribution as DistSparseMatrix<T> and is meant to be simple to use.

Constructs width vectors, each of the given height, over a particular communicator.
